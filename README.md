#Project Overview

#Problem Statement

BCA students frequently rely on shared notes from seniors and peers, but this material is scattered across WhatsApp groups, Google Drive folders and printouts. Searching for a specific subject or semester is slow, links expire, and there is no way to know which notes are actually useful. There is no central, well-organised, student-owned platform to host and discover this content.

#Proposed Solution

NoteShare provides a dedicated web application where any student can browse a catalog of notes, filter by subject and semester, search by title, and preview PDFs in-browser before downloading. Logged-in students can contribute new notes through a simple upload form. Each note carries metadata (subject, semester, description, file size, downloads), making discovery fast and ranking implicit through usage.
Target Users

• BCA students (1st to 6th semester) looking for study material.
• Senior students who want to share notes back with juniors.
• Faculty who wish to point students to community-curated resources.
Scope
In-scope: user signup/login, uploading PDF/DOCX (≤20 MB), browsing & filtering, in-browser PDF preview, download counter, light/dark theme.
Out-of-scope (current phase): admin moderation panel, comments/ratings, AI summarisation, mobile native apps.
